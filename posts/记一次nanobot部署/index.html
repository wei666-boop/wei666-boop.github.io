前情提要:最近香港大学开源了一个类似于openclaw的超级简化版本,看到很多人都在部署openclaw,于是我就决定部署nanobot,主要是部署这个简单啊

项目地址:nanobot

要求:node版本>=18 python环境

开始部署

下载项目

我直接使用的pip安装

pip install nanobot-ai

配置项目

nanobot onboard #生成配置文件

配置文件的位置在~/.nanobot/config.json

接入通讯软件

这里我使用的是Discord,原本想要使用telegram的,结果发现telegram被我删了,在登录还要收我1美元

接入Discord的步骤如下:

1.进入开发者平台

2.点击创建应用按钮并且复制令牌

3.在机器人设置中,启用消息内容意图

4.在Discord设置中选择高级,启用开发者模式,然后左键头像并复制ID

5.在配置文件中进行配置

{
  "channels": {
    "discord": {
      "enabled": true,
      "token": "YOUR_BOT_TOKEN",
      "allowFrom": ["YOUR_USER_ID"]
    }
  }
}

6.在开发者平台的Oauth中滑到底部有一个URL生成器,在上面的配置中,选择Bot,权限选择Send Messages,Read Message History,打开生成的链接并邀请进入服务器

7.执行nanobot gateway
